
# 自行构建使用

> rust 1.70+

测试通过: linux mint21.3 / win11 / macos13


# Setup-1 build

```sh
cargo build --release
``` 

# Setup-2 register

> 自行替换路径

```sh
# windows
.\target\release\typora-crack.exe C:\app\Typora

# linux
sudo ./target/release/typora-crack /usr/share/typora
```
